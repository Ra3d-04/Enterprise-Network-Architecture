# Enterprise-Network-Architecture


 Project Overview
This project demonstrates the design and deployment of a scalable multi-branch enterprise network. Simulated using Cisco Packet Tracer, the focus of this project is on establishing reliable inter-branch connectivity, implementing efficient IPv4 subnetting, and configuring centralized network servers to support organizational operations.

 Technologies & Protocols Used
* **Network Simulator:** Cisco Packet Tracer
* **Routing & Switching:** Inter-VLAN Routing, Static/Dynamic Routing, IPv4 Subnetting
* **Services:** DHCP, DNS, Server Configuration
* **Infrastructure:** Multi-router and multi-switch topology connecting distinct branch locations

##  Key Features & Configurations
* **Scalable Infrastructure:** Designed a comprehensive multi-branch network architecture utilizing efficient IP addressing and subnetting to conserve IP space.
* **Inter-Branch Connectivity:** Configured routing protocols to ensure seamless and reliable communication between different physical branch locations.
* **Server Deployment:** Configured central servers (such as DHCP and DNS) to manage and automate network services across the different branches.
* **VLAN Segmentation:** Segmented internal traffic using VLANs to optimize network performance and organize departmental traffic.
